Project Title

Motorola Mobile Sales Performance Dashboard using Power BI

Business Problem

Businesses generate large volumes of sales data every day, making it difficult to monitor performance and identify trends. Without a centralized dashboard, managers cannot easily track sales, customer preferences, payment methods, or product performance. This project addresses that challenge by providing an interactive dashboard for better business analysis and decision-making.

Short Description

This project is an interactive Power BI dashboard that analyzes Motorola mobile sales data. It provides insights into sales performance, customer ratings, payment methods, city-wise sales, monthly trends, and product performance. The dashboard helps users understand business performance through clear and interactive visualizations.

Purpose

-Monitor overall sales performance.

-Analyze sales trends across different months.

-Identify top-performing cities and mobile models.

-Understand customer purchasing behavior.

-Compare payment methods used by customers.

-Support data-driven business decisions.

Technologies Used

-Power BI Desktop

-Microsoft Excel

-Power Query

-DAX (Data Analysis Expressions)

-Data Modeling

-Interactive Filters and Slicers

Charts Used

-KPI Cards (Total Sales, Total Quantity, Transactions, Average Sales)

-Treemap (Total Sales by City)

-Line Chart (Total Quantity by Month)

-Funnel Chart (Customer Ratings)

-Pie Chart (Transactions by Payment Method)

-Bar Chart (Total Sales by Mobile Model)


-Line Chart (Total Sales by Day Name)

-Table (Brand-wise Sales Summary)

-Slicers (Month, Mobile Model, Brand, Payment Method, Day Name)

Features

-Interactive dashboard with dynamic filters

-City-wise sales analysis

-Monthly sales trend analysis

-Mobile model performance comparison

-Customer rating analysis

-Payment method analysis

-Brand-wise sales summary

-KPI cards for quick business insights

-Cross-filtering across all visuals

-Clean and user-friendly dashboard design

Highlights

Built an interactive business dashboard using Power BI.
Performed data cleaning and transformation using Power Query.
Created KPIs using DAX measures.
Analyzed sales by city, month, mobile model, and payment method.
Designed an executive-level dashboard for business reporting.
Enabled interactive filtering for better data exploration.
Demonstrated practical Business Analytics and Data Visualization skills.

Business Insights

Delhi and Mumbai generated the highest sales.
Sales quantity was highest in March and July.
UPI was the most preferred payment method.
Most customers gave 4-star and 5-star ratings.
Some mobile models consistently achieved higher sales than others.
Daily sales trends can help businesses plan promotions and inventory more effectively.
